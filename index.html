Welcome to Zar

Zar is a fun (I hope), minimal scripting language with a retro style, built using Python. Why is it called ZAR? I reversed the word "ram" and changed the M to a Z because my nickname is Żurek (by the way, that's the name of a traditional Polish soup).

Features

  • Simple syntax
  • Built-in GUI support
  • Custom interpreter written in Python

Basics
Variables 
    // setting string variable "text" to "Some text"
    set str text : "Some text"
    // setting integer variable "number" to 17
    set int number : 17
    // setting integer variable "LengthOfText" to the length of variable "text"
    set int LengthOfText : $ text
"Hello, World!" Program 
    set str msg : "Hello, World!"
    set int MsgLen : $ msg
    add MsgLen, 1
    write msg, MsgLen
Compare 
    set str msg : "x == y!"
    set int MsgLen : $ msg
    add MsgLen, 1
    set int x : 1
    set int y : 1
    compare x | y
    write msg, MsgLen
What's Your Name (Input) 
    set str ASK_MSG : "What is your name?"
    set int ASK_MSG_LEN : $ ASK_MSG
    set str msg : "name"
    write ASK_MSG, ASK_MSG_LEN
    read msg
    set str hello_msg : "Hello, "
    add hello_msg, msg
    add hello_msg, "!"
    set int MsgLen : $ hello_msg
    add MsgLen, 1
    write hello_msg, msg, MsgLen
Jump Loops 
    set str msg : "Hello";
    set int count : 0;
    set int count_end : 3;
    
    loop:
    write msg, 5;
    add count, 1;
    compare count | count_end;
    jmp end;
    jmp loop;
    
    end:
    set str msg : "end";
    write msg, 3;
Simple Calculator 
    // messages
    set str MSG1 : "Select first number."
    set str MSG2 : "Select second number."
    set str MSG3 : "Select operator (+,-,*,/)"
    
    set int MSG1LEN : $ MSG1
    set int MSG2LEN : $ MSG2
    set int MSG3LEN : $ MSG3
    
    write MSG1, MSG1LEN
    read num1
    write MSG2, MSG2LEN
    read num2
    write MSG3, MSG3LEN
    read op
    
    set int num1 : /num1
    set int num2 : /num2
    
    compare op | "+"
    jmp plus
    compare op | "-"
    jmp minus
    compare op | "*"
    jmp multiplication
    compare op | "/"
    jmp division
    jmp end
    
    plus:
        add num1, num2
        jmp end
    
    minus:
        sub num1, num2
        jmp end
    
    multiplication:
        mul num1, num2
        jmp end
    
    division:
        div num1, num2
    
    end:
        set str output : /num1
        set int len : $ output
        write output, len
File Management
Reading Files 
// setting variable msg to text from file "text file.txt"
set str msg : file("text file.txt")

set int MsgLen : $ msg
sub MsgLen, 1

write msg, MsgLen
Making Files 
// if file "src/text file.txt" already exist then text in it will be changed
set str text : "new text"
file.new "src/text file.txt" | text
Adding New Line 
// this code will add new line to file "cool file.py"
set str text : "print('Hello, World!')"
fileln "cool file.py" | text

OS
cls/clear 
// importing os library
import os

set str msg : "Hello 1"
set int MsgLen : $ msg
write msg, MsgLen

// clearing screen (works only on windows, use "clear" for linux)
cls

set str msg : "Hello 2"
set int MsgLen : $ msg
write msg, MsgLen
Windows+Linux "clear screen" Command 
// importing os and platform libraries
import os
import platform

set str msg : "Hello 1"
set int MsgLen : $ msg
write msg, MsgLen

// this command clears screen regardless of the operating system (windows/linux)
platform.system(clear screen)

set str msg : "Hello 2"
set int MsgLen : $ msg
write msg, MsgLen

Making Label 
import customtkinter

set int x : 200
set int y : 200
set str msg : "Hello, World!"

window.create(400 | 400, "GUI Labels")

//          id  x   y  text
label.create(1, x | y, msg)

window.mainloop
Changing Labels Text 
import customtkinter
// to use "label.change_after" you need to import time
import time

set int x : 100
set int y : 60
set str msg : "Waiting..."
set str msg2 : "Changed!"

window.create(400 | 300, "Auto Update Label")
label.create(1, x | y, msg)

// changing label id  text  after x time
label.change_after(1, msg2, 3)

window.mainloop
